== Installation ==

1. Upload `gtmetrix_website-performance` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit Appearance > Widgets and add the widget to your sidebar or widgetbar
